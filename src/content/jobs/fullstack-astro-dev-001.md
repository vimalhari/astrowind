## Role overview

You’ll build fast, SEO-friendly websites and web applications using Astro, modern UI patterns, and reliable backend integrations. This is a full‑stack role with a strong emphasis on performance, maintainability, and clean developer experience.

## What you’ll work on

- Astro components and page architecture (content-driven + marketing sites)
- Integrations with APIs, databases, and headless CMS
- UI implementation using TypeScript and Tailwind CSS
- Performance and SEO improvements (Core Web Vitals, accessibility)
- Deployments and environment configuration (Netlify/Vercel/AWS as needed)

## What success looks like

- You ship features that are fast, accessible, and easy to evolve
- You keep PRs small, readable, and backed by clear reasoning
- You proactively identify performance and SEO opportunities

## Interview process

1. Intro call (experience + expectations)
2. Technical discussion (Astro, TypeScript, architecture, debugging)
3. Small exercise (time-boxed) or pairing session on a real-world scenario
4. Final conversation (team fit + offer)

## How to apply

Send your resume and links to 1–2 relevant projects (GitHub or live sites). If possible, include:

- What you built and your role
- Any performance work you’ve done (CWV, Lighthouse, caching)
- Your preferred stack for backend integrations
